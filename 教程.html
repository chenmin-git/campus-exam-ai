管理员教程

1. 用户、班级、课程、授课管理

  1. 使用 admin / 123456 登录。
  2. 进入左侧“系统管理”，依次维护用户、班级、课程和授课安排。
  3. 新增用户时填写账号、角色、班级、联系方式；密码留空时使用默认初始密码。
  4. 授课安排用于绑定教师、课程和班级，学生考试列表会按班级对应课程过滤。

2. 权限管理

  1. 进入“权限管理”。
  2. 选择管理员、教师或学生角色。
  3. 勾选该角色允许访问的菜单。
  4. 点击“保存权限”，对应用户重新登录后生效。

3. 操作日志与备份恢复

  1. 进入“操作日志”，查看用户保存、删除、组卷、提交、阅卷、申诉等关键操作。
  2. 进入“备份恢复”，点击“创建备份记录”。
  3. 需要恢复时点击对应记录的“恢复”。当前实现为逻辑备份恢复记录，生产环境可继续接入 mysqldump 文件备份。

教师教程

1. 题库管理与 Excel 导入

  1. 使用 teacher / 123456 登录，进入“题库管理”。
  2. 点击“新增题目”，可选择单选、多选、判断、简答、编程题。
  3. 为题目填写课程、题干、答案、解析、难度、知识点和审核状态。
  4. 点击“导入xls”可上传 Excel 题库。模板路径：docs/tutorial-assets/import-template.xlsx。
  5. 点击“查重”可检查同课程同题干的重复题目。

Excel 列顺序：课程ID、题型、题干、A、B、C、D、正确答案、解析、难度、知识点。

2. 试卷管理与智能组卷

  1. 进入“试卷管理”，点击“新增试卷”。
  2. 选择课程，填写试卷名称、考试时长、是否发布、是否允许重考。
  3. 设置考试开始时间和结束时间，学生只能在开放时间内进入考试。
  4. 勾选题目并设置每题分值，保存后生成试卷。
  5. 点击“AI出卷”，输入题目数、总分、知识点和题型比例，系统会生成试卷草稿。

3. 成绩管理与人工阅卷

  1. 进入“成绩管理”，查看学生提交记录、总分、客观题分、主观题分和复核状态。
  2. 如果试卷包含简答题或编程题，学生提交后状态为待复核。
  3. 进入“人工阅卷”，查看学生答案，填写得分和教师批注，点击“提交”。
  4. 阅卷完成后，学生会收到复核通知。

4. 成绩分析、考试监控、成绩复查

  1. 进入“成绩分析”，查看均分、最高分、最低分、及格率和薄弱知识点。
  2. 进入“考试监控”，查看切屏、失焦、全屏异常等事件。
  3. 进入“成绩复查”，处理学生申诉，可同意或拒绝并写入回复。

学生教程

1. 在线考试

  1. 使用 student / 123456 登录，进入“在线考试”。
  2. 查看可参加考试，确认考试时长、总分、开始时间、结束时间和重考规则。
  3. 点击“开始考试”，系统创建考试记录并显示倒计时。
  4. 单选、判断题使用单选框；多选题使用复选框；简答题和编程题使用文本框作答。
  5. 考试期间可进入全屏。系统会记录切屏、失焦、全屏异常和超时提交事件。
  6. 点击“提交试卷”，客观题自动评分，主观题等待教师复核。

2. 历史成绩和错题解析

  1. 进入“历史成绩”，查看每次考试的成绩、状态、开始时间和提交时间。
  2. 进入“错题解析”，查看错题、学生答案、正确答案和原始解析。
  3. 点击“AI解析”，系统调用讯飞星火生成错因分析；接口失败时使用本地兜底解析。

3. 学习建议、成绩申诉、通知中心

  1. 进入“学习建议”，系统根据错题知识点生成 AI 复习计划和练习建议。
  2. 进入“成绩申诉”，选择考试记录，填写申诉原因并提交。
  3. 进入“通知中心”，查看考试发布、提交成功、阅卷完成、异常提醒等通知。

系统描述和代码描述

1. 系统定位

本系统是一个基于 Spring Boot 3、Vue 3、MyBatis-Plus、MySQL 和讯飞星火 API 的校园智能在线考试平台。系统围绕“教务管理、教师出卷、学生考试、自动评分、主观题阅卷、成绩分析、AI 错题复盘”形成闭环。

2. 技术架构

层级技术职责
前端Vue 3、Vite、Pinia、Vue Router、Element Plus角色菜单、业务页面、表格、表单、考试作答、截图中所有界面。
后端Spring Boot 3、MyBatis-Plus、Java 17登录鉴权、题库、试卷、考试、阅卷、统计、监控、申诉、通知。
数据库MySQL保存用户、班级、课程、题库、试卷、考试记录、答案、日志、通知等数据。
AI 能力讯飞星火 Chat Completions APIAI 出题、AI 组卷、AI 错题解析、AI 学习建议。

4. 核心业务说明

功能关键代码说明
登录与权限AuthService.java、RoleGuard.java、router/index.js后端校验角色，前端按权限控制菜单。
考试流程ExamService.java、StudentController.java控制考试时间、防重复、开始考试、提交、自动评分。
题库与导入QuestionService.java、TeacherView.vue支持题型扩展、Excel 导入、查重、审核。
组卷PaperService.java、AiAuthoringService.java手动组卷和 AI 约束组卷。
人工阅卷ExamService.manualGrade教师给主观题打分并写批注，系统重算总分。
成绩分析ExamService.analysisReport统计均分、最高分、最低分、及格率、题目正确率和薄弱知识点。
考试监控ExamMonitorEvent、/api/student/monitor记录切屏、失焦、全屏异常和 IP、浏览器信息。
申诉复查GradeAppeal、reviewAppeal学生提交申诉，教师处理并通知学生。
通知和日志NotificationService.java、OperationLogService.java记录消息提醒和关键操作审计。
