# 开源发布清单

发布到 Gitee/GitHub 前建议按下面顺序检查。

## 1. 敏感信息

- 确认没有提交真实数据库密码、讯飞星火 API Key、服务器地址、学生隐私数据。
- `backend/src/main/resources/application-local.yml` 已加入 `.gitignore`。
- 推荐使用环境变量配置：

```bash
export DB_USERNAME="root"
export DB_PASSWORD="your_password"
export SPARK_API_PASSWORD="your_spark_api_password"
```

## 2. 构建产物

不要提交这些目录：

```text
frontend/node_modules/
frontend/dist/
backend/target/
logs/
.playwright-mcp/
```

如果这些文件已经被 git 跟踪，执行：

```bash
git rm -r --cached frontend/node_modules frontend/dist backend/target logs .playwright-mcp
git rm --cached backend/src/main/resources/application-local.yml
```

## 3. 发布前验证

```bash
cd backend
mvn -q -DskipTests package

cd ../frontend
npm run build
```

## 4. 文档入口

- `README.md`：项目介绍、启动方式、截图、相关项目推荐。
- `教程.html`：完整系统使用教程。
- `docs/功能清单.md`：角色功能清单。
- `docs/tutorial-assets/`：教程截图、测试报告、Excel 导入模板。
- `CONTRIBUTING.md`：贡献指南。
- `SECURITY.md`：安全说明。
- `LICENSE`：MIT 协议。
